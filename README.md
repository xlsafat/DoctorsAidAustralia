# 🩺 Doctors Aid Australia

![Project Status](https://img.shields.io/badge/Status-Active-success)
![Platform](https://img.shields.io/badge/Platform-Static_HTML-blue)
![License](https://img.shields.io/badge/License-Colorlib-orange)

Doctors Aid Australia is a dedicated humanitarian platform designed to facilitate aid and donations for vulnerable communities. This website serves as a hub for various charitable programs including emergency food relief, medical assistance, and clean water projects.

---

## 🚀 Key Features

-   **Modern Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.
-   **Seamless Donations:** Integrated with **Stripe Payment Links** for secure, one-click contributions.
-   **Program Showcases:** Detailed sections for various aid projects with status updates.
-   **Interactive UI:** Smooth scroll animations, carousels, and stylized components.
-   **Contact Management:** PHP-powered contact form for direct inquiries.

---

## 🛠 Technology Stack

| Layer | Technologies |
| :--- | :--- |
| **Frontend** | HTML5, CSS3 (Sass), JavaScript (jQuery) |
| **Styling** | [Bootstrap 4](https://getbootstrap.com/), [Animate.css](https://animate.style/) |
| **Components** | Slick Slider, Owl Carousel, Magnific Popup |
| **Backend** | PHP (Form Handling) |
| **Payments** | [Stripe](https://stripe.com/) |

## 💳 Stripe Integration

The site uses a simplified Stripe integration via **Payment Links**.
-   **Simple Maintenance:** No backend database required.
-   **Security:** Payments are handled on Stripe's secure servers.
-   **Flexibility:** Easily update links in the HTML to change donation targets or amounts.

## 🛠 Installation & Local Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/DoctorsAidAustralia.git
    ```
2.  **Open locally:**
    Simply open `index.html` in any modern browser.
3.  **PHP Functionality:**
    To test the `contact_process.php` functionality, use a local server like **XAMPP**, **MAMP**, or the PHP built-in server:
    ```bash
    php -S localhost:8000
    ```
